## 📄 Printing
The CV is optimized for A4 paper. To save as PDF:
1. Open `index.html` in any modern browser.
2. Press `Ctrl + P` (or `Cmd + P` on Mac).
3. Select "Save as PDF" and ensure "Background Graphics" is enabled for the best result.
